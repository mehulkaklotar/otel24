 PHPWeby Free ip2country module for PHP

 What is it?
 -----------

 PHPWeby Free ip2country (ip to country) module for PHP is PHP class for resolving IP address to
 country. Useful for, displaying country name from where the visitor is, 
 web analytics, or banning users from certain countries. 
 There are other uses too, we will leave that to your imagination.

 This product uses GeoLite data created by MaxMind, available from
 http://maxmind.com/
